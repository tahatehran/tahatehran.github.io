IronClaw یک چارچوب عامل هوش مصنوعی مبتنی بر Rust است که توسط NEAR AI توسعه یافته و ایزولاسیون WebAssembly و امنیت Trusted Execution Environment (TEE) را به عامل‌های سازگار با OpenClaw می‌آورد.

## 🔩 IronClaw چیست؟

IronClaw ایمنی حافظه Rust را با مدل ایزولاسیون WASM ترکیب می‌کند تا عامل‌های هوش مصنوعی بسازد که:
- **اسکیل‌های غیرقابل اعتماد** را در sandboxهای ایمن حافظه اجرا کند
- **رمزها را در vaultهای رمزنگاری‌شده TEE** ذخیره کند
- **در برابر prompt injection** با مسیریابی مبتنی بر گراف دفاع کند
- **روی NEAR AI Cloud** برای استقرار غیرمتمرکز اجرا شود

## 🔐 قابلیت‌های امنیتی

### ۱. ایزولاسیون WASM
هر اسکیل در یک sandbox WebAssembly اجرا می‌شود:
```rust
let skill = WasmSkill::load("excel-analyzer.wasm")?;
let result = skill.execute(SkillContext {
    memory_limit: 64 * 1024 * 1024,
    network_access: false,
    file_access: FilePolicy::ReadOnly("/data"),
})?;
```

### ۲. Vaultهای رمزنگاری‌شده TEE
رمزها در enclavesهای Intel SGX/ARM TrustZone ذخیره می‌شوند:
```rust
let vault = TeeVault::new(VaultConfig {
    attestation: true,
    seal_policy: SealedToThisNode,
    rotation: Duration::from_secs(3600),
})?;
vault.store("api-key", sensitive_key).await?;
```

### ۳. دفاع در برابر Prompt Injection
```
ورودی کاربر → مسیریاب گراف → تشخیص injection → خروجی ایمن
```

## 🚀 شروع کار

```bash
# نصب ابزار Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# کلون و ساخت IronClaw
git clone https://github.com/nearai/ironclaw
cd ironclaw
cargo build --release

# اجرا با گیت‌وی OpenClaw
./target/release/ironclaw --gateway http://localhost:18789
```
