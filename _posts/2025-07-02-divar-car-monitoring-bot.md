## معرفی پروژه

**سیستم هوشمند پایش و اعلان آگهی‌های خودرو در دیوار** یک ربات هوشمند در پیام‌رسان Bale است که به صورت خودکار آگهی‌های خودرو در سایت دیوار را پایش کرده و آگهی‌های جدید را لحظه‌ای به کاربران اعلان می‌دهد.

🔗 **ربات Bale:** [@Divar_iranbot](https://ble.ir/Divar_iranbot)

---

## 🔍 ویژگی‌های برجسته

### وب اسکرپر هوشمند

- **پردازش دو مرحله‌ای:** استخراج لیست آگهی‌ها و سپس جزئیات کامل هر آگهی
- **استخراج جامع اطلاعات:**
  - 📝 عنوان و توضیحات کامل
  - 💰 قیمت (با فیلتر قیمت توافقی)
  - 📏 کارکرد دقیق
  - 🎨 رنگ خودرو
  - 📅 سال ساخت/مدل
  - 📍 محل دقیق آگهی
  - 🖼️ تصویر خودرو
- **کنترل همزمانی:** محدودیت تعداد درخواست‌های همزمان
- **Rate Limiting:** احترام به محدودیت‌های سایت دیوار

### 💬 ربات Bale هوشمند

- **احراز هویت امن:** سیستم ورود با نام کاربری و رمز عبور
- **اعلان لحظه‌ای:** ارسال آگهی‌های جدید با تصویر
- **پنل تنظیمات کاربر:**
  - ⏰ تنظیم فاصله زمانی اعلان‌ها
  - 🔔 فعال/غیرفعال کردن اعلان‌ها
  - 🏙️ انتخاب شهرهای مورد نظر
  - 📊 مشاهده آگهی‌های اخیر
- **سیستم اشتراک:** مدیریت زمان اعتبار کاربران
- **پنل مدیریت:** کنترل کامل برای ادمین‌ها

### 🔐 API امن و قدرتمند

- **احراز هویت API Key:** حفاظت از endpoint‌ها
- **مستندات خودکار:** Swagger UI و ReDoc
- **پشتیبانی از Pagination:** مدیریت حجم بالای داده
- **پاسخ‌های استاندارد:** JSON Response با کدهای مناسب

### ⚡ عملکرد بالا

- **برنامه‌ریزی هوشمند:** APScheduler برای وظایف دوره‌ای
- **Async/Await:** پردازش غیرهمزمان با httpx
- **صف اعلان‌ها:** مدیریت بهینه ارسال پیام‌ها
- **کش داده:** بهینه‌سازی عملکرد پایگاه داده
- **نگهداری خودکار:** پاکسازی هفتگی آگهی‌های قدیمی

## 📖 راهنمای استفاده

### استفاده از ربات Bale

#### 🔐 ورود به سیستم

1. ربات را در Bale پیدا کنید
2. دستور `/start` را ارسال کنید
3. نام کاربری و رمز عبور خود را وارد کنید

#### 👤 دستورات کاربری

| دستور                   | توضیحات                                               |
| ----------------------- | ----------------------------------------------------- |
| `/settings`             | نمایش پنل تنظیمات و راهنما                            |
| `/toggle_notifications` | فعال/غیرفعال کردن اعلان‌ها                            |
| `/set_interval <دقیقه>` | تنظیم فاصله زمانی اعلان‌ها (مثال: `/set_interval 15`) |
| `/get_latest`           | دریافت ۵ آگهی جدید                                    |
| `/list_car_models`      | مشاهده خودروهای در حال پایش                           |
| `/list_cities`          | مشاهده لیست شهرهای قابل انتخاب                        |
| `/my_cities`            | مشاهده شهرهای انتخابی شما                             |
| `/add_city <شهر>`       | افزودن شهر (مثال: `/add_city tehran`)                 |
| `/remove_city <شهر>`    | حذف شهر از لیست                                       |
| `/feedback <پیام>`      | ارسال بازخورد به ادمین                                |
| `/logout`               | خروج از حساب کاربری                                   |

## 🛠️ تکنولوژی‌ها

- **زبان برنامه‌نویسی:** Python
- **پردازش غیرهمزمان:** httpx + Async/Await
- **برنامه‌ریزی وظایف:** APScheduler
- **پیام‌رسان:** Bale API
- **پایگاه داده:** SQLite / PostgreSQL
- **مستندسازی API:** Swagger UI + ReDoc

## 🎯 نتیجه‌گیری

این سیستم با ترکیب وب اسکرپینگ هوشمند و ربات پیام‌رسان Bale، تجربه‌ای بی‌نظیر از پایش خودکار آگهی‌های دیوار را برای کاربران فراهم می‌کند. با رابط کاربری ساده و تنظیمات قابل شخصی‌سازی، هر کاربری می‌تواند آگهی‌های مورد نظر خود را به صورت لحظه‌ای دریافت کند.
