IronClaw is a Rust-based AI agent framework developed by NEAR AI that brings WebAssembly sandboxing and Trusted Execution Environment (TEE) security to OpenClaw-compatible agents.

## 🔩 What is IronClaw?

IronClaw combines Rust's memory safety with WASM's isolation model to create AI agents that can:
- **Execute untrusted skills** in memory-safe sandboxes
- **Store secrets** in TEE-encrypted vaults
- **Defend against prompt injection** with graph-based routing
- **Run on NEAR AI Cloud** for decentralized deployment

## 🏗️ Architecture

```
IronClaw Runtime
├── WASM Sandbox + TEE Vault + Prompt Defense
├── Rust Core Engine (Memory Safe, Async, Typed)
└── OpenClaw Compatibility (Gateway, Skills, Models)
```

## 🔐 Security Features

### 1. WASM Sandboxing
Every skill runs in a WebAssembly sandbox:
```rust
let skill = WasmSkill::load("excel-analyzer.wasm")?;
let result = skill.execute(SkillContext {
    memory_limit: 64 * 1024 * 1024, // 64MB
    network_access: false,
    file_access: FilePolicy::ReadOnly("/data"),
})?;
```

### 2. TEE Encrypted Vaults
Secrets are stored in Intel SGX/ARM TrustZone enclaves:
```rust
let vault = TeeVault::new(VaultConfig {
    attestation: true,
    seal_policy: SealedToThisNode,
    rotation: Duration::from_secs(3600),
})?;
vault.store("api-key", sensitive_key).await?;
```

### 3. Prompt Injection Defense
```
User Input → Graph Router → Detects injection → Safe Output
```

## 🚀 Getting Started

```bash
# Install Rust toolchain
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Clone and build IronClaw
git clone https://github.com/nearai/ironclaw
cd ironclaw
cargo build --release

# Run with OpenClaw gateway
./target/release/ironclaw --gateway http://localhost:18789
```
