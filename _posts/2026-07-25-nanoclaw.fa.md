NanoClaw یک عامل هوش مصنوعی سبک و ایزوله‌شده با container است که OpenClaw را به واتساپ، تلگرام، دیسکورد و پلتفرم‌های پیام‌رسان دیگر متصل می‌کند.

## 📦 NanoClaw چیست؟

NanoClaw یک گیت‌وی حداقلی (~15 فایل) Node.js است که:
- **پلتفرم‌های پیام‌رسان** را به سیستم اسکیل OpenClaw متصل می‌کند
- **هر اسکیل را در یک container** برای ایزولاسیون اجرا می‌کند
- **از agent swarm** برای اجرای موازی پشتیبانی می‌کند
- **جلسات را در کانال‌های مختلف** مدیریت می‌کند

```
گیت‌وی NanoClaw
├── واتساپ / تلگرام / دیسکورد
├── Runtime Container (اسکیل A، B، C)
└── هسته OpenClaw (مدل‌ها، حافظه، ابزارها)
```

## 🔑 قابلیت‌های کلیدی

### ۱. پشتیبانی چند کاناله
| پلتفرم | وضعیت | ویژگی‌ها |
|---------|-------|---------|
| واتساپ | ✅ پایدار | متن، تصاویر، صدا، اسناد |
| تلگرام | ✅ پایدار | Bot API، کوئری‌های inline، گروه‌ها |
| دیسکورد | ✅ پایدار | سرورها، DM، ترد، واکنش‌ها |
| سیگنال | 🔶 بتا | پیام‌رسان رمزنگاری‌شده E2E |
| اسلک | 🔶 بتا | ورک‌اسپیس‌ها، کانال‌ها، تردها |

### ۲. ایزولاسیون Container
هر اسکیل در container جداگانه Docker اجرا می‌شود:
```yaml
services:
  skill-excel:
    image: nanoclaw/skill-excel:latest
    isolation:
      network: none
      filesystem: read-only
      memory: 256MB
      cpu: 0.5
```

### ۳. Agent Swarms
استقرار چند عامل هوش مصنوعی به‌صورت موازی:
```json
{
  "swarm": {
    "agents": 5,
    "strategy": "work-stealing",
    "shared_memory": true,
    "max_parallel": 3
  }
}
```

## 🚀 شروع سریع

```bash
# نصب NanoClaw
npm install -g nanoclaw

# راه‌اندازی با کانال‌های پیام‌رسان
nanoclaw init --channels whatsapp,telegram,discord

# شروع گیت‌وی
nanoclaw start --port 18791
```

### Docker Compose
```yaml
version: '3.8'
services:
  nanoclaw:
    image: nanoclaw/gateway:latest
    ports:
      - "18791:18791"
    volumes:
      - ./config:/app/config
      - /var/run/docker.sock:/var/run/docker.sock
    environment:
      - OPENCLAW_URL=http://openclaw:18789
```

## 📊 عملکرد

| معیار | مقدار |
|-------|-------|
| راه‌اندازی سرد | < ۲ ثانیه |
| حافظه (پایه) | ~30MB |
| حافظه هر اسکیل | ~50MB |
| حداکثر همزمانی | ۵۰ جلسه |
| تأخیر پیام | < 500ms |

## 💡 موارد استفاده

- **پشتیبانی مشتری:** ربات واتساپ با دانش OpenClaw
- **همکاری تیمی:** ربات دیسکورد برای بازبینی کد و مستندسازی
- **دستیار شخصی:** ربات تلگرام با دسترسی کامل فایل
- **آموزش:** سیستم آموزشی چند پلتفرمی
- **فروش:** واجد شرایط کردن خودکار سرنخ‌ها در کانال‌ها
