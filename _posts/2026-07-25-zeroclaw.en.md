ZeroClaw is an ultra-efficient Rust framework for building AI agents that need maximum performance with minimum resource usage. With <5MB RAM, <10ms startup, and 30+ integrations, it's built for production at scale.

## 🎯 What is ZeroClaw?

ZeroClaw is a Rust framework that:
- **Uses < 5MB of RAM** at idle
- **Starts in < 10ms** from cold boot
- **Supports 30+ messaging integrations**
- **Provides SQLite-based persistent memory**
- **Includes a typed skill system** with compile-time safety

## 📊 Performance Benchmarks

| Metric | ZeroClaw | OpenClaw | NanoClaw |
|--------|----------|----------|----------|
| RAM (idle) | < 5MB | ~100MB | ~30MB |
| Startup | < 10ms | ~500ms | ~2s |
| Throughput | 20K req/s | 1K req/s | 500 req/s |
| Binary size | ~2MB | ~50MB | ~15MB |
| Latency (p99) | < 1ms | ~10ms | ~5ms |

## 🏗️ Architecture

```
ZeroClaw Framework (Rust)
├── Async Runtime (Tokio)
├── Provider Layer (30+ APIs)
├── Skill System (Typed, Compile-Time Safe)
├── Memory (SQLite + Embeddings)
└── Integrations (30+ Channels)
```

## 🔧 Getting Started

```bash
# Install ZeroClaw
cargo install zeroclaw

# Or download binary
curl -fsSL https://zeroclaw.dev/install.sh | bash

# Quick start
zeroclaw init --name my-agent
zeroclaw run
```

### Configuration
```toml
[agent]
name = "my-agent"
model = "gpt-4"
personality = "helpful and precise"

[provider]
type = "openai"
api_key = "sk-..."

[memory]
backend = "sqlite"
path = "./memory.db"

[integrations]
telegram = { token = "..." }
discord = { token = "..." }
whatsapp = { enabled = true }

[skills]
directory = "./skills"
sandbox = true
```

## 🔐 Security Features

- **Memory-safe Rust:** No buffer overflows, no null pointers
- **Skill sandboxing:** Each skill runs in isolation
- **Encrypted memory:** Sensitive data encrypted at rest
- **Audit logging:** All actions logged for compliance
