ZeroClaw یک چارچوب فوق‌کارآمد Rust برای ساخت عامل‌های هوش مصنوعی است که بیشترین عملکرد را با کمترین مصرف منابع نیاز دارند. با کمتر از 5MB RAM، کمتر از 10ms راه‌اندازی و بیش از ۳۰ ادغام، برای تولید در مقیاس ساخته شده است.

## 🎯 ZeroClaw چیست؟

ZeroClaw یک چارچوب Rust است که:
- **کمتر از 5MB RAM** در حالت بیکار استفاده می‌کند
- **کمتر از 10ms** از راه‌اندازی سرد شروع می‌شود
- **بیش از ۳۰ ادغام پیام‌رسان** پشتیبانی می‌کند
- **حافظه پایدار مبتنی بر SQLite** ارائه می‌دهد
- **یک سیستم اسکیل تایپ‌شده** با ایمنی زمان کامپایل دارد

## 📊 بنچمارک عملکرد

| معیار | ZeroClaw | OpenClaw | NanoClaw |
|-------|----------|----------|----------|
| RAM (بیکار) | < 5MB | ~100MB | ~30MB |
| راه‌اندازی | < 10ms | ~500ms | ~2s |
| عملکرد | 20K req/s | 1K req/s | 500 req/s |
| اندازه باینری | ~2MB | ~50MB | ~15MB |
| تأخیر (p99) | < 1ms | ~10ms | ~5ms |

## 🔧 شروع کار

```bash
# نصب ZeroClaw
cargo install zeroclaw

# یا دانلود باینری
curl -fsSL https://zeroclaw.dev/install.sh | bash

# شروع سریع
zeroclaw init --name my-agent
zeroclaw run
```

### پیکربندی
```toml
[agent]
name = "my-agent"
model = "gpt-4"
personality = "کمکی و دقیق"

[provider]
type = "openai"
api_key = "sk-..."

[memory]
backend = "sqlite"
path = "./memory.db"

[integrations]
telegram = { token = "..." }
discord = { token = "..." }
whatsapp = { enabled = true }
```

## 💡 موارد استفاده

- **بات‌های پرتراffic:** مدیریت هزاران کاربر همزمان
- **استقرار Edge:** اجرا روی سرورهای محدود منابع
- **SaaS چندمستاجری:** سرویس‌دهی به مشتریان متعدد از یک نمونه
- **پردازش بلادرنگ:** مدیریت پیام با تأخیر کم
- **بهینه‌سازی هزینه:** حداقل‌سازی هزینه محاسبات ابری
