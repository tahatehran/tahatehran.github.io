BirdClaw یک فضای کاری Twitter/X محلی‌اول است که کنترل کامل داده‌های Twitter شما را فراهم می‌کند. با جستجوی کامل متن SQLite FTS5، وارد کردن آرشیو و رابط‌های CLI و وب، ابزار قدرتمندی برای مدیریت Twitter است.

## 🐦 BirdClaw چیست؟

BirdClaw یک برنامه محلی‌اول است که:
- **تمام داده‌ها را به‌صورت محلی** در SQLite با جستجوی FTS5 ذخیره می‌کند
- **آرشیوهای Twitter** را برای دسترسی کامل تاریخی وارد می‌کند
- **رابط‌های CLI و وب** برای دسترسی انعطاف‌پذیر ارائه می‌دهد
- **از زمان‌بندی توییت** و عملیات انبوه پشتیبانی می‌کند
- **تجزیه و تحلیل** و ردیابی تعامل دارد

## 🔍 قابلیت‌های کلیدی

### ۱. جستجوی کامل متن (FTS5)
```sql
-- جستجو در تمام توییت‌های شما
SELECT * FROM tweets WHERE content MATCH 'openclaw tutorial';

-- جستجو با رتبه‌بندی
SELECT *, rank FROM tweets 
WHERE content MATCH 'AI agent' 
ORDER BY rank;
```

### ۲. وارد کردن آرشیو
```bash
# وارد کردن خروجی داده Twitter
birdclaw import ~/Downloads/twitter-archive.zip

# بررسی وارد کردن
birdclaw stats
```

### ۳. داشبورد تحلیل
| معیار | موجود |
|-------|-------|
| تاریخچه توییت‌ها | ✅ آرشیو کامل |
| نرخ تعامل | ✅ به‌ازای هر توییت |
| تغییرات فالوور | ✅ ردیابی روزانه |
| عملکرد هشتگ | ✅ تجمیع شده |
| بهترین زمان ارسال | ✅ تحلیل با AI |

## 🚀 شروع کار

```bash
# نصب BirdClaw
curl -fsSL https://birdclaw.sh/install.sh | bash

# راه‌اندازی فضای کاری
birdclaw init

# وارد کردن آرشیو Twitter
birdclaw import ~/Downloads/twitter-archive.zip

# شروع رابط وب
birdclaw web --port 3000
```

## 💡 موارد استفاده

- **تولیدکنندگان محتوا:** زمان‌بندی و تحلیل عملکرد توییت
- **محققان:** تحلیل روندهای Twitter و مکالمات
- **برندها:** ردیابی اشاره‌ها و تعاملات
- **شخصی:** جستجو در تاریخچه کامل توییت‌ها
- **روزنامه‌نگاران:** منبع‌یابی و بررسی اطلاعات
