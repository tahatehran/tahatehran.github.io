NullClaw یک رانتایم عامل هوش مصنوعی مبتنی بر Zig است که در فقط 678KB جای می‌گیرد و یکی از سبک‌ترین عامل‌های کامل‌قابلیت موجود است.

## 🪶 NullClaw چیست؟

NullClaw یک رانتایم عامل هوش مصنوعی از صفر نوشته‌شده با Zig است که:
- **کمتر از ۲ms راه‌اندازی** می‌شود
- **کمتر از 1MB حافظه** در حالت بیکار استفاده می‌کند
- **بیش از ۵۰ ارائه‌دهنده مدل** پشتیبانی می‌کند (OpenAI، Anthropic، Google، Ollama و...)
- **۱۹ کانال پیام‌رسان** متصل می‌شود (واتساپ، تلگرام، دیسکورد و...)
- **یک سیستم اسکیل کامل** با hot-reloading دارد

## 📊 مقایسه اندازه

| رانتایم | اندازه باینری | مصرف RAM | زمان راه‌اندازی |
|---------|-------------|----------|----------------|
| **NullClaw** | **678KB** | **< 1MB** | **< 2ms** |
| OpenClaw (Node.js) | ~50MB | ~100MB | ~500ms |
| NanoClaw | ~15MB | ~30MB | ~2s |
| IronClaw | ~5MB | ~5MB | ~10ms |

## 🚀 شروع کار

```bash
# دانلود NullClaw
curl -fsSL https://nullclaw.dev/install.sh | bash

# یا ساخت از سورس
git clone https://github.com/nullclaw/nullclaw
cd nullclaw
zig build -Doptimize=ReleaseFast

# اجرا
./nullclaw --config config.toml
```

### پیکربندی حداقلی
```toml
[provider]
type = "openai"
api_key = "sk-..."

[channels]
telegram = { token = "..." }
discord = { token = "..." }

[skills]
directory = "./skills"
hot_reload = true
```

## 💡 موارد استفاده

- **دستگاه‌های IoT:** اجرای عامل‌های هوش مصنوعی روی Raspberry Pi، ESP32 و...
- **سیستم‌های تعبیه‌شده:** خودروها، کنترل‌کننده‌های صنعتی
- **Edge Computing:** استنتاج محلی با تأخیر کم
- **توسعه:** نمونه‌سازی و آزمایش سریع
- **تولید:** استقرارهای چندمستاجری با تراکم بالا
