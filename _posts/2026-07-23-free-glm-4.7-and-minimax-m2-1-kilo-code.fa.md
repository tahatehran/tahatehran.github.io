Kilo Code اکنون دو مدل کدنویسی هوش مصنوعی در سطح فرانتیر را به صورت رایگان ارائه می‌دهد: GLM 4.7 از Z.ai و MiniMax M2.1. اگر می‌خواهید کمک کدنویسی هوش مصنوعی قوی را بدون پرداخت هزینه توکن داشته باشید، این مدل‌های رایگان به شما دسترسی به کیفیت سطح تفکر را در همان افزونه‌ای که برای Claude، GPT و Gemini استفاده می‌کنید می‌دهند.

## GLM 4.7 و MiniMax M2.1 چه چیزی متفاوت دارند؟

بیشتر ابزارهای کدنویسی هوش مصنوعی رایگان شما را به مدل‌های کوچک یا قدیمی محدود می‌کنند. GLM 4.7 و MiniMax M2.1 متفاوت هستند زیرا به عنوان مدل‌های در سطح فرانتیر با پنجره‌های context بزرگ و استدلال قوی برای کارهای مهندسی واقعی طراحی شده‌اند.

- **GLM 4.7**: Flagship مدل Z.ai با پنجره context 202K توکن. کدبس‌های عظیم، پرامپت‌های طولانی و بازنویسی چندپرونده را بدون گم شدن ساختار پروژه مدیریت می‌کند.
- **MiniMax M2.1**: مدل کارایی بالا که در بنچمارک‌های کدنویسی با مدل‌های پرداخت شده برتر رقابت می‌کند. به خصوص در وظایف پیاده‌سازی، دیباگ و توضیحات کد قوی است.

هر دو مدل در Kilo Code همراه با ۵۰۰+ مدل دیگر در دسترس هستند، بنابراین می‌توانید هنگام تغییر جریان کار بین مدل‌های رایگان و پرداخت شده جابجا شوید.

## چرا مدل‌های رایگان سطح فرانتیر مهم هستند؟

کدنویسی گران است اگر طوال روز از مدل‌های پرداخت شده فرانتیر استفاده کنید. یک بازنویسی پیچیده، یک جلسه دیباگ طولانی یا یک بررسی کامل کد می‌تواند هزاران توکن مصرف کند. مدل‌های رایگان موانع هزینه را برای کار روزمره حذف می‌کنند و همچنان کمک‌های با کیفیت ارائه می‌دهند.

این برای موارد زیر مفید است:

- دانشجوی یا مشارکت‌کننده منبع-باز که نمی‌تواند قبول API پرداخت را توجیه کند.
- توسعه‌دهنده در تیمی با بودجه ابزار محدود.
- کسی که می‌خواهد برای وظایف سبک مانند مستندات، commit یا توضیحات کمک هوش مصنوعی داشته باشد بدون سوزاندن اعتبارات پرداخت شده.

## چگونه شروع کنیم

۱. Kilo Code را در VS Code، IDE JetBrains یا CLI نصب کنید.
۲. یک حساب رایگان Kilo Cloud در app.kilo.ai ایجاد کنید.
۳. مدل GLM 4.7 یا MiniMax M2.1 را از انتخاب‌گر مدل انتخاب کنید.
۴. با کمک کدنویسی رایگان هوش مصنوعی شروع به کدنویسی کنید.

می‌توانید هر زمان بین مدل‌های رایگان و پرداخت شده جابجا شوید. هیچ قفل‌بندی نیست و می‌توانید برای گزینه‌های بیشتر کلیدهای API خود را بیاورید.

## Kilo Code به عنوان پلتفرم

Kilo Code فقط یک رابط چت نیست. پلتفرم مهندسی عامل هوش مصنوعی همه‌در-یک با حالت‌های Ask، Code، Debug و Architect است. مدل‌های رایگان در هر حالتی کار می‌کنند، بنابراین می‌توانید از آن‌ها برای سوالات، تولید کد، دیباگ و برنامه‌ریزی استفاده کنید.

از آنجایی که Kilo Code منبع-باز است، می‌توانید کد را بررسی، مشارکت یا fork کنید. افزونه روی Windows، macOS و Linux کار می‌کند و تنظیمات شما را بین دستگاه‌ها هماهنگ می‌کند.

## محدودیت‌ها

مدل‌های رایگان با مدل‌های پرداخت شده فرانتیر یکسان نیستند. GLM 4.7 و MiniMax M2.1 قوی هستند، اما ممکن است در سخت‌ترین وظایف استدلالی در مقایسه با Claude Opus یا GPT-5 عملکرد ضعیف‌تری داشته باشند. اگر جریان کار شما به استدلال برتر، طراحی معماری یا دیباگ سطح-سیستم وابسته باشد، یک مدل پرداخت شده را به عنوان پشتیبان نگه دارید.

به مدل‌های رایگان به عنوان یک پیش‌فرض محکم برای توسعه روزمره فکر کنید، نه جایگزین برای هر مدل پرداخت شده در هر موقعیتی.
