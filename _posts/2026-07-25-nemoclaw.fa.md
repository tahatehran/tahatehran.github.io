NemoClaw لایه امنیتی سازمانی NVIDIA برای OpenClaw است. این ابزار محافظت‌ها، فیلترهای حریم خصوصی و قابلیت‌های انطباق را اضافه می‌کند تا OpenClaw برای استفاده سازمانی و دولتی ایمن شود.

## 🔒 NemoClaw چیست؟

NemoClaw بین OpenClaw و داده‌های سازمانی شما قرار می‌گیرد و به‌عنوان یک دروازه امنیتی عمل می‌کند:

- **فیلتر کردن داده‌های حساس** قبل از رسیدن به مدل هوش مصنوعی
- **اجبار سیاست‌های انطباق** (GDPR، HIPAA، SOC2)
- **ثبت تمام تعاملات** برای مسیرهای بازرسی
- **مسدود کردن حملات injection** در لحظه
- **تأیید خروجی‌های هوش مصنوعی** با زمینه‌بینی بصری

## 🏢 قابلیت‌های سازمانی

### ۱. محافظ حریم خصوصی داده
```yaml
privacy:
  redact_pii: true
  mask_emails: true
  mask_phone_numbers: true
  block_ssn_patterns: true
  custom_patterns:
    - pattern: "internal-project-\\d+"
      action: redact
```

### ۲. موتور انطباق
| مقررات | محافظت |
|--------|--------|
| GDPR | تشخیص PII، حق حذف |
| HIPAA | فیلتر PHI، ثبت بازرسی |
| SOC2 | کنترل دسترسی، رمزگذاری |
| CCPA | پوشش داده، ردیابی رضایت |

### ۳. دفاع در برابر Prompt Injection
NemoClaw از NeMo Guardrails NVIDIA استفاده می‌کند:
- تشخیص promptهای تهاجمی
- مسدود کردن تلاش‌های jailbreak
- اعمال محدودیت موضوع
- تأیید دقت خروجی

## 🚀 شروع کار

```bash
# نصب NemoClaw در کنار OpenClaw
docker run -d \
  --name nemoclaw \
  -p 18790:18790 \
  -v ~/.openclaw:/root/.openclaw \
  nvcr.io/nvidia/nemoclaw:latest

# پیکربندی OpenClaw برای استفاده از NemoClaw
openclaw config set security.gateway http://localhost:18790
```

## 📊 امنیت در مقابل عملکرد

| تنظیم | سطح امنیت | تأثیر بر تأخیر |
|-------|-----------|----------------|
| **حداقلی** | فیلتر پایه PII | +5ms |
| **استاندارد** | PII + دفاع injection | +15ms |
| **سخت‌گیرانه** | موتور انطباق کامل | +30ms |
| **حداکثری** | تمام بررسی‌ها + بازرسی انسانی | +100ms |

## 💡 موارد استفاده

- **بهداشت و درمان:** پرسش‌های سازگار با HIPAA درباره داده‌های بیمار
- **مالی:** تراکنش‌های تنظیم‌شده SOX/PCI-DSS
- **دولتی:** مدیریت داده‌های طبقه‌بندی‌شده
- **حقوقی:** حفاظت از رازداری وکیل-موکل
- **منابع انسانی:** مدیریت حریم خصوصی کارکنان

## 🔮 آینده
NVIDIA به گسترش قابلیت‌های NemoClaw با چارچوب‌های انطباق جدید و تشخیص تهدید پیشرفته ادامه می‌دهد. ادغام با سیستم اسکیل OpenClaw به کاربران سازمانی اجازه می‌دهد جریان‌های کاری سفارشی بسازند.
