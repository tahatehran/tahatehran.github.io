ClawHub is the official skill registry and marketplace for the OpenClaw ecosystem. Think of it as npm for AI agent skills — a place to discover, install, publish, and share capabilities for your OpenClaw assistant.

## 🏪 What is ClawHub?

ClawHub is a centralized registry that:
- **Hosts thousands of skills** for OpenClaw
- **Provides semantic search** to find the right skill
- **Manages versioning** and dependency resolution
- **Scores trust** based on community reviews and audits
- **Supports private registries** for enterprise use

## 🔍 Discovery Features

### 1. Semantic Vector Search
```bash
# Find skills by description
clawhub search "generate PDF reports from Excel data"

# Find by category
clawhub search --category office --sort popularity

# Find by compatibility
clawhub search --runtime docker --min-trust 4.5
```

### 2. Trust Scoring
| Score | Meaning | Badges |
|-------|---------|--------|
| 4.5-5.0 | Excellent | Verified, Audited, Popular |
| 3.5-4.4 | Good | Community Tested |
| 2.5-3.4 | Fair | Use with Caution |
| < 2.5 | Poor | Not Recommended |

### 3. Version Management
```bash
# Install a specific version
clawhub install office-excel@2.3.1

# Update all skills
clawhub update --all

# Rollback
clawhub rollback office-excel --to 2.2.0
```

## 🚀 Getting Started

```bash
# Install ClawHub CLI
npm install -g clawhub-cli

# Login to your account
clawhub login

# Search for skills
clawhub search "data analysis"

# Install a skill
clawhub install office-excel

# Publish your own skill
clawhub publish ./my-skill --name my-custom-skill
```

## 📦 Skill Packages

Each skill package includes:
```
my-skill/
├── manifest.json     # Metadata, dependencies, config
├── index.js          # Main entry point
├── tests/            # Test suite
├── README.md         # Documentation
└── LICENSE           # License file
```

### manifest.json
```json
{
  "name": "office-excel",
  "version": "2.3.1",
  "description": "Excel file generation and analysis",
  "author": "openclaw-community",
  "runtime": "node",
  "permissions": ["files:read", "files:write"],
  "dependencies": {
    "xlsx": "^0.18.0"
  }
}
```

## 💡 Use Cases

- **Discovery:** Find the right skill for any task
- **Quality Assurance:** Trust scoring ensures reliability
- **Enterprise:** Private registries for internal skills
- **Community:** Share and collaborate on skill development
- **Education:** Pre-built skills for learning environments
