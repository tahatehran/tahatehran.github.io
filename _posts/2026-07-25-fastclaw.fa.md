FastClaw یک رانتایم چند عاملی سبک نوشته‌شده با Go است که شخصیت‌ها، حافظه، اسکیل‌ها و ایزولاسیون sandbox را برای اجرای همزمان چندین عامل هوش مصنوعی فراهم می‌کند.

## ⚡ FastClaw چیست؟

FastClaw یک باینری Go است که:
- **چندین عامل** با شخصیت‌های متفاوت اجرا می‌کند
- **حافظه مشترک** بین تیم‌های عامل ارائه می‌دهد
- **بازار اسکیل** برای گسترش قابلیت‌ها دارد
- **هر عامل را در sandbox ایزوله** می‌کند
- **تعویض داغ عامل‌ها** بدون از کار افتادن پشتیبانی می‌کند

## 🔑 قابلیت‌های کلیدی

### ۱. ارکستراسیون چند عاملی
```yaml
agents:
  - name: analyst
    personality: "تمرکز بر داده، دقیق، تحلیلی"
    model: gpt-4
    skills: [data-analysis, visualization]
  
  - name: writer
    personality: "خلاق، واضح، جذاب"
    model: claude-3
    skills: [content-writing, editing]
  
  - name: reviewer
    personality: "انتقادی، جزئیات‌گرا، کامل"
    model: gpt-4
    skills: [code-review, testing]
```

### ۲. حافظه مشترک
```json
{
  "memory": {
    "backend": "redis",
    "namespace": "fastclaw-team",
    "ttl": "7d",
    "sharding": "consistent-hash"
  }
}
```

## 🚀 شروع کار

```bash
# نصب FastClaw
go install github.com/fastclaw-ai/fastclaw@latest

# یا دانلود باینری
curl -fsSL https://fastclaw.dev/install.sh | bash

# اجرا با پیکربندی پیش‌فرض
fastclaw start --config agents.yaml

# بررسی وضعیت عامل‌ها
fastclaw status
```

## 💡 موارد استفاده

- **تیم‌های محتوا:** خط لوله نویسنده + ویرایشگر + بازبین
- **پشتیبانی مشتری:** عامل‌های غربالگری + متخصص + ارتقاء
- **توسعه:** عامل‌های کدنویس + تستر + استقراردهنده
- **تحقیق:** عامل‌های جستجوگر + تحلیلگر + گزارشگر
- **فروش:** عامل‌های سرنخ‌یاب + واجد شرایط + نهایی‌کننده

## 🔮 آینده
FastClaw در حال کار بر پروتکل‌های مذاکره عامل به عامل، داشبورد نظارت بصری و مقیاس‌پذیری خودکار است.
