NemoClaw is NVIDIA's enterprise-grade security enhancement for OpenClaw. It adds guardrails, privacy filters, and compliance features to make OpenClaw safe for corporate and government use.

## 🔒 What is NemoClaw?

NemoClaw sits between OpenClaw and your enterprise data, acting as a security gateway that:

- **Filters sensitive data** before it reaches the AI model
- **Enforces compliance policies** (GDPR, HIPAA, SOC2)
- **Logs all interactions** for audit trails
- **Blocks prompt injection** attacks in real-time
- **Provides visual grounding** to verify AI outputs

## 🏢 Enterprise Features

### 1. Data Privacy Guard
```yaml
privacy:
  redact_pii: true
  mask_emails: true
  mask_phone_numbers: true
  block_ssn_patterns: true
  custom_patterns:
    - pattern: "internal-project-\\d+"
      action: redact
```

### 2. Compliance Engine
| Regulation | Protection |
|-----------|------------|
| GDPR | PII detection, right to erasure |
| HIPAA | PHI filtering, audit logging |
| SOC2 | Access controls, encryption |
| CCPA | Data masking, consent tracking |

### 3. Prompt Injection Defense
NemoClaw uses NVIDIA's NeMo Guardrails to:
- Detect adversarial prompts
- Block jailbreak attempts
- Enforce topic boundaries
- Validate output accuracy

### 4. Audit & Monitoring
```
NemoClaw Gateway → Filter PII → Log All Calls → Alert On Risk
```

## 🚀 Getting Started

```bash
# Install NemoClaw alongside OpenClaw
docker run -d \
  --name nemoclaw \
  -p 18790:18790 \
  -v ~/.openclaw:/root/.openclaw \
  nvcr.io/nvidia/nemoclaw:latest

# Configure OpenClaw to use NemoClaw
openclaw config set security.gateway http://localhost:18790
```
