PicoClaw is a Go-based ultra-lightweight AI agent designed to run on $10 hardware like Raspberry Pi, RISC-V boards, and other resource-constrained devices. It brings the full OpenClaw experience to the edge.

## 🦔 What is PicoClaw?

PicoClaw is a minimal Go binary that:
- **Runs on $10 hardware** (Raspberry Pi Zero, Sipeed LicheeRV, etc.)
- **Uses < 10MB of RAM** and boots in under 1 second
- **Supports Telegram, Discord, WhatsApp** out of the box
- **Includes persistent memory** with SQLite
- **Runs skills** in sandboxed goroutines

## 🚀 Getting Started

```bash
# Install PicoClaw
curl -fsSL https://picoclaw.dev/install.sh | bash

# Or build from source
git clone https://github.com/sipeed/picoclaw
cd picoclaw
go build -o picoclaw .

# Run on any device
./picoclaw --config config.yaml
```

### Raspberry Pi Setup
```bash
# On Raspberry Pi OS
sudo apt update && sudo apt install golang sqlite3
git clone https://github.com/sipeed/picoclaw
cd picoclaw
make pi-setup
make run
```
