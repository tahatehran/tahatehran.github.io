PicoClaw یک عامل هوش مصنوعی فوق‌سبک مبتنی بر Go است که برای اجرا روی سخت‌افزار ۱۰ دلاری مانند Raspberry Pi، بردهای RISC-V و دستگاه‌های محدود منابع طراحی شده است.

## 🦔 PicoClaw چیست؟

PicoClaw یک باینری حداقلی Go است که:
- **روی سخت‌افزار ۱۰ دلاری** اجرا می‌شود (Raspberry Pi Zero، Sipeed LicheeRV و...)
- **کمتر از 10MB حافظه** استفاده می‌کند و کمتر از ۱ ثانیه راه‌اندازی می‌شود
- **از تلگرام، دیسکورد، واتساپ** به‌صورت پیش‌فرض پشتیبانی می‌کند
- **حافظه پایدار** با SQLite دارد
- **اسکیل‌ها را در goroutineهای sandbox اجرا** می‌کند

## 📊 سازگاری سخت‌افزار

| دستگاه | قیمت | RAM | وضعیت PicoClaw |
|--------|------|-----|----------------|
| Raspberry Pi Zero 2W | $15 | 512MB | ✅ پشتیبانی کامل |
| Sipeed LicheeRV | $10 | 64MB | ✅ پشتیبانی کامل |
| ESP32-S3 | $5 | 8MB | ⚠️ حالت حداقلی |
| Orange Pi Zero 3 | $20 | 1GB | ✅ پشتیبانی کامل |

## 🚀 شروع کار

```bash
# نصب PicoClaw
curl -fsSL https://picoclaw.dev/install.sh | bash

# یا ساخت از سورس
git clone https://github.com/sipeed/picoclaw
cd picoclaw
go build -o picoclaw .

# اجرا روی هر دستگاهی
./picoclaw --config config.yaml
```

### راه‌اندازی Raspberry Pi
```bash
sudo apt update && sudo apt install golang sqlite3
git clone https://github.com/sipeed/picoclaw
cd picoclaw
make pi-setup
make run
```

## 💡 موارد استفاده

- **اتوماسیون خانگی:** دستیار هوشمند برای خانه‌های هوشمند
- **ایستگاه هواشناسی:** تحلیل و هشدار هوای محلی
- **دوربین امنیتی:** تشخیص حرکت با هوش مصنوعی
- **monitoring کشاورزی:** تحلیل سلامت محصول
- **کیت‌های آموزشی:** آموزش هوش مصنوعی روی سخت‌افزار ارزان

PicoClaw در حال اضافه کردن پشتیبانی از شتاب‌دهنده‌های TPU، اتصال LoRaWAN و بازار مهارت‌های بهینه‌شده edge است.
