ClawHub رجیستری رسمی اسکیل‌ها و بازارگاه اکوسیستم OpenClaw است. آن را به‌عنوان npm برای مهارت‌های عامل هوش مصنوعی در نظر بگیرید — مکانی برای کشف، نصب، انتشار و اشتراک‌گذاری قابلیت‌ها.

## 🏪 ClawHub چیست؟

ClawHub یک رجیستری متمرکز است که:
- **هزاران اسکیل** برای OpenClaw میزبانی می‌کند
- **جستجوی معنایی** برای یافتن اسکیل مناسب ارائه می‌دهد
- **مدیریت نسخه** و حل وابستگی انجام می‌دهد
- **امتیاز اعتماد** بر اساس بررسی‌های جامعه و ممیزی‌ها ارائه می‌دهد
- **از رجیستری‌های خصوصی** برای استفاده سازمانی پشتیبانی می‌کند

## 🔍 قابلیت‌های کشف

### ۱. جستجوی برداری معنایی
```bash
# یافتن اسکیل‌ها بر اساس توضیحات
clawhub search "تولید گزارش PDF از داده‌های Excel"

# یافتن بر اساس دسته‌بندی
clawhub search --category office --sort popularity

# یافتن بر اساس سازگاری
clawhub search --runtime docker --min-trust 4.5
```

### ۲. سیستم امتیاز اعتماد
| امتیاز | معنی | نشان‌ها |
|--------|------|---------|
| 4.5-5.0 | عالی | تأیید شده، ممیزی شده، محبوب |
| 3.5-4.4 | خوب | تست شده توسط جامعه |
| 2.5-3.4 | قابل قبول | با احتیاط استفاده کنید |
| < 2.5 | ضعیف | توصیه نمی‌شود |

## 🚀 شروع کار

```bash
# نصب CLI ClawHub
npm install -g clawhub-cli

# ورود به حساب
clawhub login

# جستجوی اسکیل‌ها
clawhub search "تحلیل داده"

# نصب یک اسکیل
clawhub install office-excel

# انتشار اسکیل خود
clawhub publish ./my-skill --name my-custom-skill
```

## 💡 موارد استفاده

- **کشف:** یافتن اسکیل مناسب برای هر کاری
- **تضمین کیفیت:** امتیاز اعتماد اطمینان از قابلیت اطمینان را می‌دهد
- **سازمانی:** رجیستری‌های خصوصی برای مهارت‌های داخلی
- **جامعه:** اشتراک‌گذاری و همکاری در توسعه مهارت‌ها
- **آموزش:** اسکیل‌های از پیش ساخته برای محیط‌های یادگیری

## 🔮 آینده
ClawHub در حال کار بر توصیه‌های هوشمند اسکیل با هوش مصنوعی، اسکن امنیتی خودکار و سازنده مهارت بصری است.
