آموزش سریع پروژه Mirror

این آموزش براساس اطلاعات مخزن movtigroup/Mirror آماده شده است.

۱) معرفی

Mirror مجموعه‌ای از مستندات و ابزارهای آینه نرم‌افزاری است که برای دسترسی سریع‌تر و پایدارتر به سرویس‌ها استفاده می‌شود.

۲) دستورات One-click

  • لینوکس (اوبونتو/دبیان): bash <(curl -sSL https://Linux.ththt.ir)
  • نصب داکر و آینه: bash <(curl -sSL https://install.ththt.ir/docker.sh)
  • فقط تنظیم آینه داکر: bash <(curl -sSL https://install.ththt.ir/mirror.sh)

۳) اجرای محلی مستندات

برای اجرای محلی پروژه:

docker-compose up -d --build

سپس آدرس http://localhost:8080 را باز کنید.

۴) لینک‌های مهم

  • مخزن GitHub: github.com/movtigroup/Mirror
  • وبسایت مستندات: doc.movtigroup.ir
