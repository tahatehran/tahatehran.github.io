# tahatehran.github.io

Landing page رسمی **MovtiGroup** روی GitHub Pages.

## Domain

- Primary domain: `movtigroup.me`
- Contact email: `info@movtigroup.me`

## ساختار پروژه

- `/index.html` صفحه اصلی با محتوای معرفی، خدمات، Blog و تماس
- `/styles.css` استایل‌های سایت
- `/script.js` اسکریپت سبک برای تم و سال فوتر
- `/blog/index.html` صفحه اصلی بلاگ
- `/blog/mirror-tutorial.html` آموزش پروژه Mirror
- `/blog/quick-tips.html` نکته‌های کاربردی کوتاه
- `/robots.txt` قوانین خزنده‌ها
- `/sitemap.xml` نقشه سایت برای موتورهای جستجو

## SEO

در `index.html` این موارد برای SEO اضافه و تنظیم شده‌اند:

- meta description, keywords, robots
- Canonical URL
- Open Graph / Twitter metadata
- Structured data (JSON-LD) برای Organization

## Deploy

این مخزن یک سایت static است و با GitHub Pages منتشر می‌شود.  
اگر سایت روی دامنه نمایش داده نشد، DNS دامنه باید به GitHub Pages تنظیم باشد.
